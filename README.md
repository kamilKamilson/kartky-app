# **Kartky - your best to-do lists**

Aplikacja obsługująca **[kartky api](https://github.com/kamilKamilson/kartky-api)** napisana w **React** przy użyciu **Vite** i **Typescript**.

## Uruchomienie aplikacji

```
npm i
npm run dev
```
